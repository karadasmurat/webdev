An HTML form is used to collect user input. The user input is most often sent to a server for processing. It is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc.

The HTML input element is the most used form element. An input element can be displayed in many ways, depending on the type attribute.

The label tag defines a label for many form elements. The for attribute of the label tag should be equal to the id attribute of the input element to bind them together.
